== Description ==
## Elespare - News Magazine and Blog Elements Addons for Elementor
A stunning assortment of WordPress post widgets for Elementor. It allows you to display your posts in many formats such as grid, list, tile, full, express, and so on..


== Frequently Asked Questions ==

= Do I need to code? =

Nope, you do not need to have coding knowledge.

= Do I need to any third party plugins? =

Definitely yes, you need install Elemetor page builder plugin and you must have latest version grater than 2.8.0.

= Do I need to design the layout? =

No you don't have to design the widget because our plugin already has a range of pre-designed gorgeous layouts from which you can select one.


== Installation ==
* Download the plugin
* Download the Elemetor (If needed)
* Upload to your plugin directory or simply install via WordPress admin interface.
* Activate.
* You will get beautiful custom widgets on page/post editor to build your site.
